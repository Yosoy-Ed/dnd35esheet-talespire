Estructura de los macros:


-- Basico --

nombre de lanzamiento:dados
Fireball : 8d6

-- Se debe usar guion de piso despues del dado para agregar modificadores, name : dice_mods --

Magic missile : 1d6_1
New Spell: 1d6_1+5

-- Se pueden usar algunos valores predefinidos ( ej: ACL es "Arcane caster level", ver al final) --

Fireball : ACLd6
Magic attack : ACLd6_INT*DEX/2

-- Se pueden usar limitadores as [valor | maximo] --

Fireball : [ACL|10]d6 >>> Maximo 10 d6
Curar Heridas leves : 1d8_[DCL|5] >>> 1d8 + nivel de lanzador, Maximo 5

-- Se pueden usar multiples lanzamientos separando con comma (,) --

Fireball : [ACL|10]d6, Resistencia a conjuros:1d20_ACL
Fireball : ACLd6, Magic attack : ACLd6_INT+3, Magic missile : 1d6_1

-- Se pueden agregar comentarios con @:Comentario despues de una coma, NO USAR COMAS EN LOS COMENTARIOS --

Curar Heridas leves : 1d8_[DCL|5], @:1d8 + nivel de lanzador Maximo 5

-- Los siguientes valores predefinidos son validos --

Fuerza Destreza Constitucion Inteligencia Sabiduria Carisma

RESTRICCIONES

- No se pueden usar sumas de dados, ej: 1d8_1d4, 1d8+1d4, 1d8-1d4_3, no funcionarian. aunque tecnicamente es posible, se hace demasiado complejo y por la forma en la que talespire agrupa lanzamientos de dados, puede generar confusiones y resultados no deseados.

- No se pueden realizar multipliacion de dados. ej: 3*(1d8_1), se deberia escribIr 3d8_3. Si se puede realizar operacion en la parte del modificador, ej: 1d8_3*INT/2

- 1d2 y 1d3 se implementaron de manera adicional ya que no hay soporte para estos dados, estos dados no generan lanzamientos, si en el macro hay mas dados soportados (d4 d6 d8 d11 d12 d20 d100), esos si generan lanzamiento. ej: 1d3_INT, muestra el resultado en el chat directamente. 1d3_INT, 1d6,@:Sneak_attack_extra_1d6_cada_dos_niveles, mostrara el resultado del d3 despues de lanzar el d6 junto al comentario.

- No se deben usar acentos en el nombre del lanzamiento
